# Travel Admin (PackPlango) - React Admin Panel

This is a simple React admin panel scaffold for a travel website.
It uses Material-UI icons/components for a modern look and separate CSS files for page/component styles.

How to run:
1. Unzip the project.
2. Run `npm install` to install dependencies.
3. Run `npm start` to start the development server.

Notes:
- This project includes dummy data only.
- Adjust or connect to your backend API endpoints as needed.
